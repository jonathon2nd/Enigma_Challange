Hello, the team that won this challenge last year, won 1st place ;)

This challenge consists of encrypted messages, using WWII Nazi enigma machines.
You will be given encrypted messages, with different clues on how to break them.

There are Army machines, which only use the first three rotors.
Army machines only use Reflectors B and C.

And the Navy Machines, which CAN more than three rotors at one time, and have many more to chose from.
